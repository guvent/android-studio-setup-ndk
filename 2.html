Örnek bir proje oluşturalım adet yerini bulsun :)
Sırasıyla resimlerdeki gibi sihirbazı kullanarak proje oluşturacağız.

Start a new Android Studio project diyoruz
 

Application Name kısmına uygulama için bir isim girin.
 

Çalışmasını istediğimiz cihaz türlerini minimum versiyon olarak seçiyoruz.
 

İstediğimiz şablon varsa seçiyor devam ediyoruz.
 

Ana Ekran için aktivite adı girin ( isteğe bağlı bence aynen kalsın:) )
 
Ve tamamdır.
 
Eğer Invalid Project JDK diye bir hata sağ üstte belirirse buraya tıklayıp problemi çözebilirsiniz.
